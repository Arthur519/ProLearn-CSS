O que é CSS?

ﾠCSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas.
O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML. Ele é uma ferramenta
muito importante que possibilita criar diversas funcionalidades ao invés de usar JavaScript ou outra linguagem mais pesada.
Se usado com moderação, CSS pode viabilizar uma ótima experiência ao desenvolvedor e usuários das páginas web.


Tags mais utilizadas:

  • color (Adiciona uma cor que pode ser especificada ao elemento.)
  • width (Largura de um elemento.)
  • border (Linha sólida do lado de fora do padding.)
  • padding (Acrescenta um espaçamento entre o elemento citado.)
  • margin (O espaço externo a um elemento.)
  • background-color (Cor atrás do conteúdo de um elemento e do padding.)
  • text-shadow (Cria uma sombra no texto dentro de um elemento.)
